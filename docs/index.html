Features

Auto-Connect

Detects the game process automatically within seconds. Launch before or after the game — no manual setup needed.

Speed

One-Click Presets

Set values to common amounts instantly. From 100 to 999 million at the tap of a button.

Lock Gold or Souls so the game cannot decrement them. Spend freely while your balance stays fixed.

How It Works

1

Launch the Editor

Download and extract. Double-click run.vbs — no console, no install. The editor opens and waits for the game process.

2

Start Far Far West

The editor auto-detects the game executable and attaches. The status indicator switches to Connected within seconds.

3

Edit Your Values

Type an amount and press Set, or tap any preset button. Gold and Souls update in real-time via direct memory access.

4

Freeze and Play

Click the freeze button to lock a value. The editor rewrites it every 50ms — the game cannot touch it while it is frozen.
